# Weather_DashBoard_UI
## About the Project
Weather Dashboard is a JavaFX-based application that provides real-time weather information using the OpenWeather API. Users can search for any city and view weather details such as temperature, humidity, wind speed, and weather conditions.

## Features
- Search weather by city name
- Real-time weather updates
- Display temperature, humidity, and wind speed
- User-friendly graphical interface
- Error handling for invalid city names

## Technologies Used
- Java
- JavaFX
- OpenWeather API
- JSON

## How to Run
1. Open the project in NetBeans.
2. Add the required JavaFX libraries.
3. Run the project.
4. Enter a city name and click the search button.
